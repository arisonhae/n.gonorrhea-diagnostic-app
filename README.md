# 스마트폰 기반 임질 진단 시스템 (N. gonorrhoeae Diagnostic App)

LAMP–CRISPR 형광 분석 결과를 스마트폰 카메라로 인식하고  
YOLOv8 기반 ROI 탐지를 통해 임질균 감염 여부를 자동 판정하는 Streamlit 앱입니다.  
Gemini API를 연동하여 결과 보고서와 상담 가이드도 제공합니다.

---

## 🚀 Features
- YOLOv8 기반 ROI 검출 (tube / roi 클래스)
- 자동 형광 강도 측정 (Il/Iu 비율)
- Gemini API 기반 진단 리포트 및 대화형 상담
- Streamlit Cloud에서 웹앱처럼 실행 가능
